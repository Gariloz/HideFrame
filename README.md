# HideFrame

**HideFrame** is an addon for World of Warcraft 3.3.5a that hides dragon frame decorations and faction/PVP icons from target, focus, player, and boss frames. It features a user-friendly Ace3-based configuration and supports both English and Russian.

## Features
- Hide dragon textures around target, focus, and boss frames
- Hide faction/PVP icons on target, focus, and player frames
- Easy configuration via Blizzard interface or `/hideframe` command
- Localization: English and Russian

## Installation
1. Copy the `HideFrame` folder to your `Interface/AddOns` directory in your WoW 3.3.5a client.
2. Make sure all Ace3 libraries are present in the `Libs` folder.
3. Enable the addon in the character selection screen.

## Usage
- Open settings via the interface menu or with `/hideframe` (`/hf`).
- Enable or disable the options you need.

## Requirements
- World of Warcraft 3.3.5a (Interface 30300)
